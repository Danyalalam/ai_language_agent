# Screenshots

Drop the following images here; the project README references them by these exact
file names. Recommended width ~1400px, PNG.

| File | What to capture |
| --- | --- |
| `hero.png` | The app on the **German Tutor** view — sidebar visible, conversation card centered. Good as the top banner. |
| `german-tutor-idle.png` | German Tutor before starting — the mic orb and "Gespräch starten" button. |
| `german-tutor-live.png` | German Tutor mid-conversation — status pill showing "Ich höre zu…" / "Spricht…" with a transcript line or two. |
| `pronunciation.png` | The Pronunciation view with a scored result (accuracy / fluency / prosody + AI report). |
| `architecture.png` | Optional: an architecture/data-flow diagram (or reuse the ASCII one in the README). |

Tip: capture at a browser width of ~1280–1440px with the window a bit taller than
default so the sidebar and the full card are both visible.
